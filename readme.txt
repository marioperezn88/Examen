el siguiente proyecto consiste en la creacion de una calculadora con la capacidad de sumar, restar, multiplicar y dividir,
además de añadirle una funcionalidad de números romanos.

Este proyecto lo hemos dividido en diferentes historias de usuario que a la vez se dividen en tareas.

Historias de usuario
    
     1. Sumar 2 numeros naturales
     2. multiplicar 2 numeros naturales
     3. restar 2 numeros naturales
     4. cuando no se puede restar
     5. numeros romanos

cada historia se divide en 3 tareas, una seria la preparacion del test unitario, otra la preparacion de la propia tarea y finalmente el refactoring, una vez tengamos el verde del test.
Al ser TDD, realizamos primeros la prueba unitaria, que bloquearia a la tarea 2, que no podria realizarse hasta terminar la primera.

indicamos las tareas prioritarias y el tiempo requerido para realizarlas:
 

    Primer sprint

    1. suma, con un tiempo de 1 semana aproximadamente -> compromiso para el primer sprint (test unitario/programacion sumar)
    2. restar, con un tiempo de 1 semana aproximadamente -> compromiso para el primer sprint (test unitario/programacion restar)
    3. resta que provoque un error, con un tiempo de 1 semana aproximadamente -> compromiso de entrega en el primer sprint (test unitario/programacion error)


    Segundo sprint

    4. multiplicacion o producto, con tiempo de 1 semana aproximadamente <> compromiso para el segundo sprint (test unitario/programacion multiplicar)
    5. numeros romanos, con tiempo de 1 semana aproximadamente -> compromiso para el segundo sprint (test unitario/programacion numeros romanos)

teniendo en cuenta los recursos disponibles para el proyecto y los integrantes del equipo, 
hemos concluido que se podra realizar el proyecto en 2 sprint de 1 semana, en la cual, en la primera se entregarían las funciones de sumar,
restar y comprobar el error de la resta, y en la segunda semana estarían listos para la entrega el resto de funciones, 
además de los numeros romanos

Visto todo esto, consideramos que el tiempo de finalizacion del proyecto es de 2 semanas
